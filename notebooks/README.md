# Jupyter Notebooks

This directory contains Jupyter notebooks for exploratory data analysis and model experimentation.

## Contents

- `fraud_detection_analysis.ipynb` - Complete EDA and model building pipeline
- `model_comparison.ipynb` - Model performance comparison and visualization
- `hyperparameter_tuning.ipynb` - Hyperparameter optimization experiments

## Usage

To run notebooks:

```bash
jupyter notebook
```

Make sure to install Jupyter and all dependencies from `requirements.txt` first.
